Buka Job Tracker di browser
Klik tombol "☁️ Sync Sheets" di navbar
Paste URL tadi ke kolom "URL Web App"
Klik "💾 Simpan URL"
Coba klik "⬆ Push ke Google Sheets" → cek Google Sheet kamu!


Cara Penggunaan Sehari-hari
Tombol	Fungsi
⬆ Push	Kirim semua data lokal → Google Sheets (overwrite)
⬇ Pull	Ambil data dari Google Sheets → merge ke lokal (tidak menghapus data)
💾 Simpan URL	Simpan URL tanpa melakukan sync
TIP

Gunakan Push setiap selesai update data. Gunakan Pull ketika buka di perangkat lain atau setelah edit langsung di Google Sheet.
