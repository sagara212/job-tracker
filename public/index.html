📖 Panduan Job Tracker

Selamat datang! 👋 Panduan ini membantu kamu menggunakan Job Tracker dari nol — tanpa perlu keahlian teknis.

1

Apa itu Job Tracker?

Aplikasi pribadi untuk mencatat semua lamaran pekerjaan yang sudah kamu kirim. Kamu bisa tahu lamaran mana yang sudah dibalas, mana yang belum, dan mana yang lolos interview — semuanya di satu tempat.

📋
Catat lamaran

Simpan semua detail: perusahaan, posisi, tanggal, gaji, dll.

📊
Pantau status

Lihat berapa lamaran yang interview atau sudah diterima.

2

Menambah Lamaran Baru

  1. Klik tombol + Tambah di pojok kanan atas.
  2. Isi form yang muncul:
    • Nama Perusahaan & Posisi — wajib diisi
    • Employment Type — Full-time, Part-time, dll.
    • Job Type — WFO, WFA, Hybrid, Remote
    • Tanggal Apply — wajib diisi
    • Priority — seberapa penting bagimu
    • Status — kondisi lamaran saat ini
    • Field lainnya opsional (gaji, link, lokasi, catatan)
  3. Klik Simpan. Lamaran langsung muncul di halaman utama!
3

Memahami Status Lamaran

Setiap lamaran punya status yang bisa diubah kapan saja:

📩 AppliedSudah kirim lamaran
😶 No ResponseBelum ada kabar
🎤 InterviewDiundang wawancara
❌ DitolakTidak diterima
🎉 AlhamdulillahDiterima! 🎊

💡 Ubah status: klik ✏️ Edit lalu ganti dropdown Status.

4

Pilihan Tampilan

Ganti tampilan data dengan tombol di toolbar:

⊞
Card

Kartu lamaran, mudah dibaca

☰
Table

Tabel, cocok layar lebar

⬚
Kanban

Kolom per status, drag & drop

≡
Timeline

Urutan waktu, mirip log

5

Filter & Cari Lamaran

  • 🔍 Kolom Cari — ketik nama perusahaan atau posisi
  • Filter Status / Priority / Tipe / Job Type — saring data sesuai kebutuhan
  • Urutkan — terbaru, terlama, priority tinggi/rendah, atau A–Z
6

Hubungkan Job Tracker ke Google Sheets agar data tersimpan aman di cloud — gratis!

1

Persiapan Google Sheets

  • Buka sheets.new untuk membuat spreadsheet baru.
  • Beri nama file sesukamu (misal: Database Job Tracker).
  • Kamu tidak perlu mengetik apapun — script akan membuat header otomatis.
2

Pemasangan Google Apps Script

  • Di menu Google Sheets, klik Extensions > Apps Script.
  • Hapus semua kode default yang ada di editor.
  • Copy dan Paste seluruh kode berikut:
/**
 * JOB TRACKER — GOOGLE APPS SCRIPT
 */
const SHEET_NAME = 'Job Tracker';
const HEADERS = [
  'ID', 'Perusahaan', 'Posisi', 'Employment Type', 'Lokasi',
  'Tanggal Apply', 'Tgl Interview', 'Gaji', 'Priority', 'Status',
  'Link', 'Requirements', 'Catatan', 'Label IDs', 'Dibuat', 'Diperbarui',
];
function getOrCreateSheet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sheet = ss.getSheetByName(SHEET_NAME);
  if (!sheet) {
    sheet = ss.insertSheet(SHEET_NAME);
    sheet.appendRow(HEADERS);
    const headerRange = sheet.getRange(1, 1, 1, HEADERS.length);
    headerRange.setFontWeight('bold').setBackground('#263238').setFontColor('#FFFFFF');
    sheet.setFrozenRows(1);
    sheet.autoResizeColumns(1, HEADERS.length);
  }
  return sheet;
}
function rowToJob(row) {
  return {
    id: row[0] || '', company: row[1] || '', position: row[2] || '',
    employmentType: row[3] || '', location: row[4] || '', applyDate: row[5] || '',
    interviewDate: row[6] || '', salary: row[7] || '', priority: row[8] || 'Medium',
    status: row[9] || 'Applied', applyLink: row[10] || '', requirements: row[11] || '',
    notes: row[12] || '', labelIds: row[13] ? JSON.parse(row[13]) : [],
    createdAt: row[14] || '', updatedAt: row[15] || '',
  };
}
function jobToRow(job) {
  return [
    job.id, job.company, job.position, job.employmentType, job.location,
    job.applyDate, job.interviewDate, job.salary, job.priority, job.status,
    job.applyLink, job.requirements, job.notes, JSON.stringify(job.labelIds || []),
    job.createdAt || new Date().toISOString(), job.updatedAt || new Date().toISOString()
  ];
}
function doGet(e) {
  try {
    const sheet = getOrCreateSheet();
    const rows = sheet.getDataRange().getValues();
    if (rows.length <= 1) return ContentService
      .createTextOutput(JSON.stringify({ success: true, jobs: [] }))
      .setMimeType(ContentService.MimeType.JSON);
    const jobs = rows.slice(1).filter(row => row[0]).map(rowToJob);
    return ContentService.createTextOutput(JSON.stringify({ success: true, jobs }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ success: false, error: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
function doPost(e) {
  try {
    const payload = JSON.parse(e.postData.contents);
    const action = payload.action || 'push';
    const jobs = payload.jobs || [];
    const sheet = getOrCreateSheet();
    if (action === 'push') {
      const lastRow = sheet.getLastRow();
      if (lastRow > 1) {
        sheet.getRange(2, 1, lastRow - 1, HEADERS.length).clearContent();
        sheet.deleteRows(2, lastRow - 1);
      }
      if (jobs.length > 0)
        sheet.getRange(2, 1, jobs.length, HEADERS.length).setValues(jobs.map(jobToRow));
      sheet.autoResizeColumns(1, HEADERS.length);
      return ContentService.createTextOutput(JSON.stringify({ success: true, written: jobs.length }))
        .setMimeType(ContentService.MimeType.JSON);
    }
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ success: false, error: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
3

Deployment Penting!

  1. Klik tombol Deploy (biru, kanan atas) > New deployment.
  2. Pilih icon gerigi (Select type) > Web app.
  3. Description: Job Tracker API.
  4. Execute as: Me (Email kamu).
  5. Who has access: Anyone (wajib agar app bisa kirim data).
  6. Klik Deploy.
  7. Saat diminta izin: pilih email > Advanced > Go to (unsafe) > Allow.
4

Hubungkan ke Job Tracker

  1. Setelah deploy berhasil, salin Web App URL.
  2. Klik tombol ☁️ Sync Sheets di navbar.
  3. Paste URL ke input yang tersedia.
  4. Klik 💾 Simpan URL.
5

Cara Pakai

⬆
Push ke Google Sheets

Kirim data dari HP/Laptop ke Sheets (overwrite).

⬇
Pull dari Google Sheets

Ambil data dari Sheets ke HP/Laptop (misal ganti HP).

🎉 Sekarang datamu aman tersimpan di Google Sheets secara gratis!
